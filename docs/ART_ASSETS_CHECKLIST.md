## 七、资源命名规范

### 通用规范
- 使用小写字母和下划线
- 使用英文命名
- 文件名具有描述性
- 同类资源统一前缀

### 前缀规范
| 前缀 | 用途 | 示例 |
|------|------|------|
| bg_ | 背景 | bg_town.png |
| btn_ | 按钮 | btn_normal.png |
| icon_ | 图标 | icon_gold.png |
| hex_ | 六边形 | hex_normal.png |
| hero_ | 英雄 | hero_castle_knight.png |
| skill_ | 技能 | skill_fireball.png |
| effect_ | 特效 | effect_fire.png |
| bgm_ | 背景音乐 | bgm_town.mp3 |
| sfx_ | 音效 | sfx_click.mp3 |
